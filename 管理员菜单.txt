========================================
          Joker 插件 - 管理员菜单
========================================

┌─────────────────────────────────────────┐
│              【菜 单】                  │
└─────────────────────────────────────────┘

├─ 【scum帮助】
│  ├─ 【普通设置】
│  │  ├─ /激活卡密 <卡密>
│  │  ├─ /绑定id <服务器ID>
│  │  ├─ /解绑
│  │  ├─ /查询在线
│  │  ├─ /id查询 <关键词>
│  │  └─ /查看授权
│  │
│  └─ 【主人指令】
│     ├─ /生成卡密 <天数> [数量]
│     ├─ /群组授权 <群组ID> [天数]
│     ├─ /取消授权 <群组ID>
│     ├─ /增加时间 <群组ID> [天数]
│     ├─ /减少时间 <群组ID> [天数]
│     ├─ /查询卡密
│     ├─ /删除已用卡密
│     ├─ /删除全部卡密
│     └─ /查看授权
│
└─ 【群管设置】
   ├─ /自定义回复
   ├─ /禁言系统
   ├─ /踢出系统
   ├─ /黑名单系统
   ├─ /入群审核
   ├─ /撤回系统
   ├─ /入群欢迎
   └─ /分群设置

提示: 输入子命令可查看详细用法，如: /自定义回复


========================================
          【自定义回复】功能说明
========================================

┌─────────────────────────────────────────┐
│          【自定义回复】                 │
└─────────────────────────────────────────┘
├─ /添加回复 <问题> <答案>    - 添加精确匹配回复
├─ /模糊添加 <问题> <答案>    - 添加模糊匹配回复
├─ /删除 <问题>              - 删除指定回复
├─ /查看回复列表              - 查看所有回复
├─ /清空回复                  - 清空所有回复
└─ /设置回复冷却 <秒数>       - 设置回复冷却时间


========================================
          【禁言系统】功能说明
========================================

┌─────────────────────────────────────────┐
│           【禁言系统】                  │
└─────────────────────────────────────────┘
├─ /禁言 @用户 [时长] [理由]  - 禁言用户（支持5s/5m/5d格式）
├─ /解除禁言 @用户              - 解除用户禁言
├─ /查看禁言列表              - 查看群内禁言列表
└─ /设置禁言级别 <级别>       - 设置禁言级别（1-10）

========================================
          【入群审核】功能说明
========================================

┌─────────────────────────────────────────┐
│          【入群审核】                   │
└─────────────────────────────────────────┘
│
├─ 【审批模式（4选1）】
│  ├─ direct  - 直接审批（管理员手动）
│  ├─ code    - 审批码审批（同意+4位码）
│  ├─ math    - 算数验证（随机加减）
│  └─ id      - ID验证（输入5位）
│
├─ 【审核管理】
│  ├─ /开启入群审核                - 开启入群审核
│  ├─ /关闭入群审核                - 关闭入群审核
│  ├─ /设置审批模式 <模式>         - 设置审批模式
│  ├─ /审核通过 <QQ号>           - 审核通过用户
│  ├─ /拒绝入群 <QQ号> [理由]     - 拒绝用户入群
│  ├─ /查看待审核列表             - 查看待审核列表
│  ├─ /添加白名单 @用户           - 添加到白名单
│  ├─ /移除白名单 @用户           - 从白名单移除
│  ├─ /设置验证超时 <秒数>         - 设置验证超时时间
│  ├─ /设置验证最大尝试 <次数>     - 设置最大尝试次数
│  ├─ /查看验证设置             - 查看验证设置
│  └─ /回答验证 <答案>            - 回答验证问题
│
├─ 【邀请统计】
│  ├─ /查看邀请排行              - 查看邀请排行榜
│  ├─ /查看邀请记录              - 查看邀请记录
│  ├─ /查看邀请记录 @用户         - 查看用户邀请记录
│  └─ /检查异常邀请 [阈值]         - 检查异常邀请
│
└─ 【操作日志】
   ├─ /查看操作日志              - 查看操作日志
   ├─ /查看操作统计              - 查看操作统计
   └─ /清空操作日志              - 清空操作日志


========================================
          【撤回系统】功能说明
========================================

┌─────────────────────────────────────────┐
│           【撤回系统】                  │
└─────────────────────────────────────────┘
│
├─ 【自动撤回设置】
│  ├─ /开启撤回                  - 开启自动撤回
│  ├─ /关闭撤回                  - 关闭自动撤回
│  ├─ /设置撤回时间 <时间>         - 设置撤回时间（如60s/5m）
│  ├─ /设置撤回模式 <模式>         - 设置撤回模式（keyword/regex/all）
│  ├─ /查看撤回关键词             - 查看撤回关键词列表
│  └─ /添加撤回白名单 @用户       - 添加撤回白名单
│
├─ 【关键词管理】
│  ├─ /添加撤回关键词 <关键词>      - 添加撤回关键词
│  └─ /删除撤回关键词 <关键词>      - 删除撤回关键词
│
└─ 【手动撤回】
   ├─ /撤回 <条数> [时间]         - 撤回最近消息（任意用户）
   └─ /撤回 @用户 <条数> [时间]    - 撤回指定用户消息

          【分群设置】功能说明
========================================

┌─────────────────────────────────────────┐
│          【分群设置】                   │
└─────────────────────────────────────────┘
│
├─ 【配置克隆】
│  ├─ /克隆全局                  - 克隆全局设置到本群

│  └─ /重置群组配置             - 重置本群配置
│

   ├─ reply      - 自定义回复
   ├─ mute       - 禁言系统
   ├─ kick       - 踢出系统
   ├─ blacklist  - 黑名单
   ├─ audit      - 入群审核
   ├─ recall     - 撤回系统
   └─ welcome    - 入群欢迎

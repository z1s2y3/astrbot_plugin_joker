# Joker插件 - SCUM查询与QQ群管

集SCUM服务器查询与QQ群管功能于一体的多功能插件。

## ✨ 功能特性

### 🎮 SCUM服务器查询
- 支持带 `/` 和不带 `/` 前缀的命令
- 服务器绑定功能，绑定后一键查询
- 简化格式显示，查看更直观
- 卡密授权系统，支持分群授权和时间控制

### 👥 QQ群管功能

| 功能模块 | 说明 |
|---------|------|
| 🔐 入群审核 | 支持算数验证、ID验证、审批码模式、直接模式 |
| ⚫ 黑名单系统 | 自动检测和踢出黑名单用户，支持退群自动拉黑、验证失败自动拉黑 |
| 🗑️ 撤回系统 | 支持关键词、正则、全部撤回模式 |
| 👋 入群欢迎 | 自定义欢迎消息，支持变量替换 |
| 📊 邀请统计 | 记录和统计群成员邀请情况 |
| 🔇 禁言系统 | 支持手动和自动禁言 |
| 🚫 踢出系统 | 支持手动踢人、连带踢人、自动拉黑 |
| 🔍 关键词过滤 | 全局关键词过滤，支持警告/禁言/踢出/拉黑 |
| 🎯 用户专属回复 | 针对特定用户的专属回复 |
| ⏰ 定时消息 | 定时发送消息功能 |
| 🧑‍💼 分群管理员 | 支持分群管理员设置 |
| 🖼️ 文转图 | 机器人所有文字回复自动转为图片 |

## 📖 使用说明

### 1. 基础查询（白名单群组）

白名单群组可以直接使用所有查询功能：

```
id查询 中国
id查询 38615532
```

### 2. 绑定服务器（管理员/群主/主人）

首次使用需要绑定服务器：

```
绑定id 38615532
```

### 3. 查询在线（绑定后）

绑定后，直接使用 `查询在线` 查看服务器状态：

```
查询在线
```

显示格式：
```
【服务器名称】
人数: 77/128
时间: 15:05
状态: 在线
区服: 中国
IP: 202.189.14.25:7052
```

### 4. 分群配置

每个群组独立存储配置和数据，位于 `group_configs/群号.json`

#### 克隆全局配置

```
/克隆全局
```

#### 启用/禁用功能

```
/启用功能 audit
/禁用功能 blacklist
```

#### 设置功能配置

```
/设置功能 audit mode math
/设置功能 welcome message 欢迎{nick}入群
```

### 5. 授权系统

#### 启用授权

在插件设置中开启 `启用授权系统`，非白名单群组需要卡密授权才能使用。

#### 生成卡密（主人操作）

```
生成卡密 30
```

#### 激活卡密（管理员操作）

```
激活卡密 JK301234567890ABCDEF12
```

## 💾 数据存储

- **分群配置**：每个群的独立配置和数据存储在 `group_configs/群号.json`
  - 包含：自定义回复、黑名单、禁言列表、邀请统计、操作日志、撤回记录等
  - 每个群使用各自的配置，互不影响
- **全局配置**：插件设置中的默认配置，用于克隆到分群
- **授权数据**：存储在 `auth_data.json`
- **群组绑定**：存储在 `group_bindings.json`

## ⚠️ 注意事项

1. 所有命令均支持带 `/` 和不带 `/` 两种方式
2. 卡密只能使用一次，重复使用会提示"该卡密已被使用过"
3. 主人账号在配置中添加，格式为纯数字ID
4. 建议修改默认的管理密钥以提高安全性
5. 分群配置独立存储，每个群使用各自的配置和数据
6. 修改全局配置后，需要使用 `/克隆全局` 同步到各群
7. 使用文转图功能需要安装 Pillow 库：`pip install pillow`

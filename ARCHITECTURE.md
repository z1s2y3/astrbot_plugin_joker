## 🎛️ 配置项 (_conf_schema.json)

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| whitelist_groups | list | [] | 白名单群组 |
| admin_groups | list | [] | 管理员群组 |
| owner_ids | list | [] | 主人账号 |
| enable_query | bool | true | 启用SCUM查询 |
| enable_binding | bool | true | 启用SCUM绑定 |
| enable_auth | bool | false | 启用授权系统 |
| enable_settings | bool | true | 启用群组设置 |
| enable_help | bool | true | 启用帮助功能 |
| default_enabled | bool | true | 默认启用查询 |
| allow_other_query | bool | true | 允许查询其他服务器 |
| allow_slash_prefix | bool | true | 启用命令功能 |
| owner_ignore_auth | bool | true | 主人不受授权限制 |
| owner_ignore_binding | bool | true | 主人不受绑定限制 |
| auth_key | string | "joker_plugin_secret_key_2024 | 管理密钥 |
| auto_delete_used_keys_days | int | 0 | 自动删除已用卡密天数 |
| qqadmin_enable_reply | bool | false | 启用自定义回复 |
| qqadmin_enable_mute | bool | false | 启用禁言系统 |
| qqadmin_enable_kick | bool | false | 启用踢出系统 |
| qqadmin_enable_blacklist | bool | false | 启用黑名单 |
| qqadmin_enable_audit | bool | false | 启用入群审核 |
| qqadmin_default_verification_type | string | "math" | 默认验证类型 |
| qqadmin_default_verification_timeout | number | 300 | 默认验证超时 |
| qqadmin_max_verify_attempts | number | 3 | 最大验证尝试 |
| qqadmin_enable_recall | bool | false | 启用撤回系统 |
| qqadmin_enable_welcome | bool | false | 启用入群欢迎 |
| qqadmin_enable_group_config | bool | false | 启用分群设置 |
| welcome_check_interval | number | 30 | 欢迎检查间隔 |
| qqadmin_default_reply_cooldown | number | 0 | 默认回复冷却 |
| qqadmin_default_mute_level | number | 5 | 默认禁言级别 |
| qqadmin_default_kick_limit | number | 3 | 默认踢出上限 |
| qqadmin_default_recall_time | number | 60 | 默认撤回时间 |
| qqadmin_default_recall_mode | string | "keyword" | 默认撤回模式 |
| qqadmin_default_welcome_delay | number | 0 | 默认欢迎延迟 |

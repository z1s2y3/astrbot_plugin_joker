## 📁 数据存储结构

```
plugin_data/auth/
├── auth.json              # 授权数据（卡密、群组授权）
├── bindings.json          # 服务器绑定数据
└── group_settings.json    # 群组设置
```

---

## 🎛️ 配置项 (_conf_schema.json)

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| whitelist_groups | list | [] | 白名单群组 |
| admin_groups | list | [] | 管理员群组 |
| owner_ids | list | [] | 主人账号 |
| enable_query | bool | true | 启用SCUM查询 |
| enable_binding | bool | true | 启用SCUM绑定 |
| enable_auth | bool | false | 启用授权系统 |
| enable_settings | bool | true | 启用群组设置 |
| enable_help | bool | true | 启用帮助功能 |
| default_enabled | bool | true | 默认启用查询 |
| allow_other_query | bool | true | 允许查询其他服务器 |
| allow_slash_prefix | bool | true | 启用命令功能 |
| owner_ignore_auth | bool | true | 主人不受授权限制 |
| owner_ignore_binding | bool | true | 主人不受绑定限制 |
| auth_key | string | "joker_plugin_secret_key_2024" | 管理密钥 |
| auto_delete_used_keys_days | int | 0 | 自动删除已用卡密天数 |
